# HireMeJA — Jamaica's #1 Online Job Board

[hiremeja.com](https://www.hiremeja.com) — Connecting employers and job seekers across Jamaica.

---

## About

HireMeJA is Jamaica's dedicated online job board built for local employers and job seekers. 
We connect talent across every industry and parish on the island — from Kingston boardrooms 
to Montego Bay resorts.

Whether you are a fresh graduate launching your career, an experienced professional ready 
for your next move, or an employer searching for Jamaica's best talent, HireMeJA makes the 
process fast, simple, and effective.

---

## Features

- Live job listings across all industries updated daily
- Candidate profiles and resume uploads
- Employer dashboard with applicant management
- Subscription packages for employers
- Jamaica salary guide
- Career resources — resume templates, interview tips, industry reports
- Get Certified — professional development directory
- Admin dashboard with SEO reporting and marketing analytics
- Firebase Authentication, Realtime Database, and Cloud Storage
- Cloud Functions for email notifications, payments, and job expiration

---

## Tech Stack

- **Frontend:** React 18, Tailwind CSS, Material UI
- **Backend:** Firebase (Auth, Realtime Database, Storage, Cloud Functions)
- **Payments:** Custom checkout flow
- **Hosting:** Firebase Hosting
- **Analytics:** Google Analytics 4, Google Search Console
